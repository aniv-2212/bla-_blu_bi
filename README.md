# CyberAI Application Layer Dashboard

A premium enterprise SOC dashboard focused on Application Layer Security, built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, Zustand, ECharts, TanStack Table, and Socket.IO-inspired mock streaming data.

## Start the project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the app at:

```text
http://localhost:5173/
```

## Build for production

```bash
npm run build
```

## Preview the production build

```bash
npm run preview
```
